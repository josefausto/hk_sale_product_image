Sequence Number & Product Images on Sale Order, Delivery Order & Invoice Report.

With this module you can easily see Product Images and Sequence Number in Sale Order, Delivery Order and Invoice Report.


Sale Order


As you can see after we install the module you can see Product Image Configuration in Sale Order, Delivery Order and Invoice.After we tick the show sequence number and show print image boolean we can see sequence no and product image in Sale Order Report.You can also set the size of the image.


As you can see Sequence No and Product Image in Quotation Report.


Suppose we set the image size as medium and you can see medium size image in report.



Suppose we set the image size as large and you can see large size image in report.




Delivery Order


As you can also see Product Image Configuration in Delivery Order.


As you can see after we tick show sequence number and show print image boolean we can see sequence no and product image in Picking Operations Report.


You can also see Sequence Number and Product Image in Delivery Slip Report.



Invoice


As you can also see Product Image Configuration in Invoice.


As you can see after we tick show sequence number and show print image boolean we can see sequence no and product image in Invoice Report.
